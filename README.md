# -Air-Quality-Monitoring-System-Using-Arduino-Uno
Arduino-based air quality, temperature, and humidity monitoring system using MQ-135 and DHT11 sensor.

**Arduino Air Quality Monitoring System Components**
• Arduino board
• MQ135 gas sensor for detecting various gases
• DHT11 temperature and humidity sensor
• OLED display for visual output
• Breadboard and jumper wires
